# fullstackexample

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Start MongoDB Server
```
mongod
```

### Start Backend
```
node server.js
```
Note: I am running the backend in a python virtual environment so that the wiki script works.
